# Modelos para AgavePredictor

Para que la aplicación funcione correctamente, coloca aquí los siguientes archivos:

1. model_agave_final.joblib - El modelo RandomForest entrenado
2. preprocessor.pkl - El preprocesador scikit-learn

Estos archivos son necesarios para realizar predicciones precisas del tiempo de crecimiento del agave.

Si no tienes estos archivos, la aplicación utilizará un modelo de demostración que simula predicciones, pero no será tan preciso como el modelo entrenado real.
